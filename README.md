# Description

This project is a machine learning-based phishing email detection system developed using Python and Scikit-learn.
The system analyzes email content and identifies whether an email is "Phishing" or "Safe" based on text patterns,
keywords, and extracted features.

The project demonstrates important cybersecurity and machine learning concepts such as text processing,
classification models, feature extraction, and email threat detection. It helps users understand how machine 
learning can be used to detect suspicious emails and reduce phishing attacks.

# Features

- Email text analysis
- Phishing email detection
- Safe email classification
- Machine learning model training
- Accuracy calculation
- Feature extraction from email content
- Detection using keywords and text patterns

# Technologies Used

- Python
- Pandas
- Scikit-learn
- CountVectorizer
- Machine Learning Algorithms
